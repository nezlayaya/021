This module extends functionality of default category module in Opencart.

It was written for Opencart version 2.0.3.1 so far. Support for other versions of Opencart will be released soon.

Installation instruction:

1. Go to Extensions -> Extension Installer in admin panel of your store.
2. Upload archive with this extension..
3. Go to Extensions -> Modifications in admin panel and click on Refresh button.
4. For styling of module functionality use modcategory.css file in "stylesheet" folder.
5. visiy our site http://fly-studio.com.ua/.

Thank you!
